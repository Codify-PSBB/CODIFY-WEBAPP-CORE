# Coding Club Competition Platform

Initial scaffold for an internal school coding competition platform.

## Stack

- Frontend: React + Vite
- Backend: Cloudflare Worker (TypeScript)
- Database: Cloudflare D1 (SQLite)
- Global state: Cloudflare KV
- Auth: Clerk
- Python runtime: Pyodide (browser only)

## Repository Structure

- `frontend/` React application scaffold
- `worker/` Cloudflare Worker scaffold
- `database/` D1 migration scaffold
- `docs/` Project documentation
- `shared/` Shared types/utilities scaffold

## Status

This commit scaffolds structure only. No application features are implemented yet.
